# Sistema de login/registro com múltiplas funcionalidades.

Em apenas duas semanas de estudos com a linguagem Java, tomei a decisão de desenvolver um projeto de estudo de Registro e Login simples, mas que contenha a validação/autenticação básica de usuário.

# Práticas essenciais desse desenvolvimento:

• Modificadores de acessos: public e private.
• Encapsulamento de funções/métodos para uma melhor legibilidade do código.
• Utilizei janelas JOptionPane em vez de System.outs.
• Prefiro desenvolver em inglês porque me sinto mais familiarizado com a linguagem — gosto pessoal. 
• Nenhuma inteligência artificial foi usada. Tudo foi desenvolvido por mim com duas semanas de estudos. Se tiver alguma dúvida, saberei responder cada uma delas.
